# Smoke Test Checklist

## Creation flow
- Sign in and submit a prompt from home page.
- Confirm immediate redirect to project builder.
- Verify step-by-step AI progress appears in sidebar and preview loader.
- Verify final website renders in preview.

## Revision flow
- Submit a revision prompt from sidebar.
- Confirm progress messages update until completion.
- Verify a new version entry appears.
- Verify failure path shows clear assistant error and loading stops.

## Publish and public view
- Toggle publish from project page.
- Verify project appears in community list.
- Open public `view` route and verify code renders.

## Persistence actions
- Save project code and refresh.
- Delete project from My Projects.
- Roll back to previous version and verify code changes.
